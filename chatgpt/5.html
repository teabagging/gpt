The algorithm should also trace and print the intermediate sequence after each step of the sorting process. Approach1. Understanding Insertion Sort: The algorithm works by building a sorted array one element at a time. For each element, it is compared with the previous elements in the sorted subarray and inserted into its correct position by shifting larger elements to the right.2. Steps: - Read the input values. - Print the initial unsorted array. - For each element starting from the second element (index 1) to the last element (index N-1): - Store the current element as the key. - Compare the key with each element in the sorted subarray (elements before the current index) and shift larger elements to the right. - Insert the key into its correct position. - Print the array after each insertion step.3. Intermediate Steps: After each insertion step, the current state of the array is printed to show the progress of the sorting process. Solution Code```pythondef insertion_sort(): n = int(input()) arr = list(map(int, input().split())) print(' '.join(map(str, arr))) for i in range(1, n): key = arr[i] j = i - 1 while j >= 0 and arr[j] > key: arr[j + 1] = arr[j] j -= 1 arr[j + 1] = key print(' '.join(map(str, arr)))insertion_sort()``` Explanation1. Reading Input: The first line of input is the number of elements `N`, followed by the sequence of integers.2. Initial Print: The initial array is printed before any sorting steps.3. The elements before the key are compared and shifted right if they are larger than the key, creating space for the key to be inserted in its correct position.5. Print Intermediate Steps: After each insertion step, the current state of the array is printed to show the progress.This approach ensures that each step of the sorting process is visible, providing a clear trace of how the array is sorted incrementally.
